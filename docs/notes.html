Notes — Fourier Transform

math
signal-processing
Published

March 26, 2026

Definition

The continuous Fourier transform of a function \(f(t)\) is:

\[ \hat{f}(\xi) = \int_{-\infty}^{\infty} f(t)\, e^{-2\pi i \xi t}\, dt \]

And the inverse:

\[ f(t) = \int_{-\infty}^{\infty} \hat{f}(\xi)\, e^{2\pi i \xi t}\, d\xi \]

Key properties

Property Time domain Frequency domain
Linearity \(\alpha f + \beta g\) \(\alpha \hat{f} + \beta \hat{g}\)
Time shift \(f(t - t_0)\) \(e^{-2\pi i \xi t_0} \hat{f}(\xi)\)
Convolution \((f * g)(t)\) \(\hat{f}(\xi) \cdot \hat{g}(\xi)\)
Parseval \(\int |f|^2\, dt\) \(\int |\hat{f}|^2\, d\xi\)

Worked example — Gaussian

For \(f(t) = e^{-\pi t^2}\), the Fourier transform is its own Gaussian:

\[ \hat{f}(\xi) = e^{-\pi \xi^2} \]

This is the only function (up to scaling) that is an eigenfunction of the Fourier transform with eigenvalue 1.

If we want it to have normalization one, we then need to define: \[ \hat{f}(\xi) = \frac{1}{\sqrt{2 \pi \sigma^2}} e^{-\pi \xi^2} \] —

Code block (optional)

You can also include executable Python code cells if you have Jupyter installed:

import numpy as np
import matplotlib.pyplot as plt

t = np.linspace(-3, 3, 1000)
f = np.exp(-np.pi * t**2)
xi = np.fft.fftfreq(len(t), d=t[1]-t[0])
fhat = np.fft.fft(f) * (t[1]-t[0])

plt.plot(np.fft.fftshift(xi), np.abs(np.fft.fftshift(fhat)))
plt.xlabel(r"$\xi$")
plt.title("Magnitude spectrum")
plt.show()
